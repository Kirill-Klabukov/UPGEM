# Changelog

All notable changes to Skill-ED will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Core Tauri/Vue application scaffold
- End-to-end encryption with ChaCha20-Poly1305
- Skill management system (create, read, update, delete)
- Habit tracking with streak monitoring
- Milestone and goal tracking
- Secure local SQLite database
- User authentication with master password
- Dark/light theme support
- Responsive UI components
- GitHub Actions CI/CD pipeline
- Comprehensive documentation
- Security policy and contributing guidelines

### Security
- Argon2id password hashing
- ChaCha20-Poly1305 AEAD encryption
- Secure random nonce generation
- Zero-knowledge architecture
- Protected sensitive data storage

## [0.1.0] - 2024-12-10

### Initial Release
- Project scaffold created
- Architecture and security foundation established
- Database schema designed
- UI framework implemented
- Documentation completed
